PROJECT OVERVIEW

About SignAI

SignAI is a custom sign recognition website built for our recorded signs. It uses the browser camera, extracts body and hand landmarks with MediaPipe, sends 15 frames to the Flask backend, and the trained LSTM model predicts one of the custom words.

This project is not an ASL translator. It only recognizes the 20 custom signs that were recorded and trained for this model.

Input

15 frames × 225 features

Tracking

Pose + left hand + right hand

Model

LSTM custom sign classifier
